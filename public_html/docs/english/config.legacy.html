Daily Quote Plugin

The Daily Quote Plugin allows the site admin and users to submit quotations for display in blocks. The administrator can control publication by using the submission queue.

This version requires GLFusion 1.1.0 or higher.

Daily Quote Plugin Configuration Options

General Settings
Option Description
Quotes per page to display The number of quotes to display per page in the main plugin.
Max search results to return Maximum quotations to be returned by the glFusion search facility.
Use submission queue? Set this to "True" to use the glFusion submission queue to moderate submissions. If this is set to "False" then user submissions will be immediately published.
Show Quotes in the "What's New" block If this is set to "Yes", then quotes which are newer than the number of days specified above will appear in the "What's New" block.
Number of days for a quote to be considered "new" This is the number of days during which a quote will appear in the "What's New" block, if enabled.
Allow anonymous users to submit quotes? If this is "Yes", then anonymous users will be able to submit quotations. If this is "No", then only logged-in users may submit. If you set this value to "Yes" then you should use the submission queue!
Allow logged-in users to submit quotes? If this is "Yes", then site users will be able to submit quotations, subject to the Submission Queue setting above.
Add Google links to quotes? Set this to "Yes" to have links to Google created for quotations. This allows quick searching for more inforamtion related to the quote.

Setting this to "No" prevents Google links from being added.
Base link for Google searches The text in this field will be used to construct a Google search link by replacing the first "%s" with the language, and the second "%s" with the search string. The default value is
http://www.google.com/search?hl=%s&num=20&q=%s, which will search Google and return 20 results.

Leave this string empty to disable link creation.
>Email admin upon quote submission? Select "Yes" to have an email sent to the site administrator when a quote is submitted.
Display glFusion Blocks Select which glFusion blocks will be displayed on the plugin home page.
CenterBlock Settings
Option Description
Set this to "Yes" to enable the Center Block. If this is "No", then none of the other CenterBlock settings will have any effect.
CenterBlock Position Select the position for the centerblock.
Show title in centerblock? Set to "Yes" to display the quotation title in the centerblock. Set to "No" to omit the title from the display.
Centerblock on Home Page only? Set to "Yes" to display the centerblock only on the home page. Set to "No" to display the centerblock on all pages.
Show source date in centerblock? If set to "Yes", the quotation's source date will be shown in the centerblock. Set to "No" to omit the source date.
Show contributed date in centerblock? If set to "Yes", the date when the quotation was submitted to your site will be shown in the centerblock. Set to "No" to omit the contribution date.
Show categories in centerblock? Set to "Yes" to display the quotations' categories in the centerblock. Set to "No" to omit categories.
Show "Add" link in centerblock? Set to "Yes" to show an "Add Quotation" link in the centerblock. Set to "No" to omit the link.
Centerblock replaces home page? Set to "Yes" to have the home page replaced by the centerblock showing a random quote.
PHP Block Settings
Option Description
Show title in PHP block? Set to "Yes" to display the quotation title in the PHP block. Set to "No" to omit the title from the display.
Show source date in PHP block? If set to "Yes", the quotation's source date will be shown in the PHP block. Set to "No" to omit the source date.
Show contributed date in PHP block? If set to "Yes", the date when the quotation was submitted to your site will be shown in the PHP block. Set to "No" to omit the contribution date.
Show categories in PHP block? Set to "Yes" to display the quotations' categories in the PHP block. Set to "No" to omit categories.
Show "Add" link in PHP block? Set to "Yes" to show an "Add Quotation" link in the PHP block. Set to "No" to omit the link.

Show source date in Regular block? If set to "Yes", the quotation's source date will be shown in the Regular block. Set to "No" to omit the source date.

Default Permissions
Option Description
Default Group This is the default group which will be used to determine who may bypass the submission queue and moderate submissions.
